Engage Static Design

Created by BD103

Hello. This website / project is to sketch out the styling and UI without any of the backend clutter. The actual project itself uses both Sass and Jinja2 to create an easy yet productive coding environment. Read more on this page, or use the navigation to look through some of the different sketches.

Project Setup

The actual project itself is set up into one main script, poetry config files, the docs folder (where the site is generated), the Sass folder, the templates folder, and the static folder. You can see the project files on Github. It currently has two external dependencies, Libsass and Jinja2.

Style Guide

Engage has a specific set of color palettes used to style webpages. When it comes to coloring, spread some slight linear gradients into the mix. Not too much to show off, but enough to shy away from material design. Always use REM for measurement, and try to add curved borders. (About 0.2 rem)

body {
  color: black;
}
